Foire au questions

Je n'arrive pas à me connecter à mon compte.

Si vous n'arrivez pas à vous connecter à votre compte CloudCore, vous pouvez utiliser l'option "mot de passe oublié" de la page de connection.

Si vous n'arrivez pas à retrouver votre identifiant, vous pouvez utiliser notre formulaire de contact en précisant dans le message la nature de votre problème.

Comment accéder au panneau de contrôle ?

Pour accéder au panneau de contrôle, connectez-vous à votre compte CloudCore, puis cliquez sur le bouton "panneau de contrôle" dans l'en tête.

Comment mettre en place une machine virtuelle ?

Pour créer une machine virtuelle, il vous faut premièrement choisir le produit le plus adapté à vos critères dans la page "produits".

Vous pourrez par la suite créer une machine virtuelle depuis le panneau de contrôle.

Vous devrez aussi paramétrer la machine suivant vos besoins.

En cas de problèmes ou de questions, vous pouvez contacter notre service client qui vous guidera et vous offrira un support personalisé.

Comment paramétrer une machine virtuelle ?

Vous devez accéder au panneau de contrôle, puis sélectionner la machine que vous souhaitez configuer.

Dans le menu se trouvera un onglet "paramètres". D'ici, vous pourrez customizer votre machine virtuelle.

Où trouver les informations sur une machine ?

Vous devez accéder au panneau de contrôle, puis sélectionner la machine dont vous souhaitez avoir les informations.

Dans le menu se trouvera un onglet "informations".

Ma machine ne démarre pas. Quoi faire ?

Si votre machine ne démarre pas correctement, vous pouvez trouver dans le log, accessible depuis le panneau de contrôle, un code d'erreur.

Celui-ci vous indiquera le problème que rencontre votre machine. Vous pourrez par la suite envoyer un message au support technique, qui vous assitera pour corriger l'erreur.
